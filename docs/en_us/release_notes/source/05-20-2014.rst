*************
edX Studio
*************

* This release introduces context-sensitive online help to Studio. You can click
  **Help** in the upper-right corner to get more more information about the
  Studio page you are viewing. You can also click links that are part of the UI
  text help.

  .. image:: /images/help.png
   :alt: Online help links in Studio

  A page in edX's continuously updated documentation that is related to the Studio page you were viewing opens in a new browser tab or window:

  .. image:: /images/help_page.png
   :alt: A Studio online help page


  You can also expand the panel at the bottom of the page and use the links at
  to access a PDF file of edX course staff documentation, as well as and other
  Studio resources.

  .. image:: /images/sock_help.png
   :alt: Other help resources at the bottom of the Studio UI
